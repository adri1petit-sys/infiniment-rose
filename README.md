# Infiniment Rose — Landing page

Landing page de la 3ᵉ édition du backyard trail du **Saint-Avertin Run Club**, au profit d'**Octobre Rose** via la Ligue contre le Cancer 37.

**Date de l'event** : samedi 10 octobre 2026, 12h00
**Lieu** : parking LaFORÊT, 35 Rue du Général Mocquery, 37550 Saint-Avertin
**Limite** : 150 coureurs · 20 € minimum · 100 % reversé

## Démarrage rapide

```bash
npm install
npm run dev
```

Puis http://localhost:3000

Voir [`SETUP.md`](./SETUP.md) pour le détail complet.

## Contexte projet

Voir [`CLAUDE.md`](./CLAUDE.md) pour le brief complet destiné à Claude Code.

## Stack

Next.js 15 · React 19 · TypeScript · Tailwind CSS 3.4 · Framer Motion 11 · Lucide Icons.
À brancher ensuite : Supabase, HelloAsso, Leaflet (GPX), Resend.
